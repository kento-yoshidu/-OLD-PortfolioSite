boxとmaskを用意する

外枠としてboxクラス、浮かび上がってくる部分としてmaskクラスを用意します。

maskの中には、最初から見えている部分のslideUpクラス、隠れている部分のtextクラスを用意します。

<div class="box">
  ここがboxクラスです
  <div class="mask">
    <p class="slideUp">Slide Up</p>
    <p class="text">Lorem ipsum dolor...略</p>
  </div>
</div>

cssは、まずは以下のように記述してみます。

* {
  margin: 0;
  padding: 0;
}

.box {
  height: 30vw;
  width: 30vw;
  border: 1px solid #aaa;
}

.mask {
  width: 100%;
  height: 100%;
  background-color: #445;
  color: #fff;
}

boxクラスはwidth、heightを指定してください（heightは%指定はできませんので、pxやvh、vwを使用してください）。また、maskクラスはboxクラスと同じ大きさになるようにしてください。

CSSを適用させると以下のように表示されるはずです。boxクラス直下に「ここがboxクラスです」という文字列を記述していますので、その分maskクラスが下にずれています。

maskクラスの位置を決定する

次にboxクラスを基準とし、maskクラスの位置を決定します。使用するCSSプロパティは**position**です。

基準となる(=親要素)boxクラスにposition: relative;、子要素にposition: absolute;を指定します。

そして、親要素を基準として、どれくらい子要素をずらすかを、top、bottom、left、rightプロパティで指定します。以下の例では`top: 70%;`、`left: 10px;`を指定しています。

(略)

.box {
  position: relative;
  height: 30vw;
  width: 30vw;
  border: 1px solid #aaa;
}

.mask {
  position: absolute;
  top: 70%;
  left: 10px;
  width: 100%;
  height: 100%;
  background-color: #445;
  color: #fff;
}

まず、何が起こったかを説明します。`top: 70%`によって、親要素であるboxクラスの上から70%分、maskが下にずれます。この70%はboxクラスの高さの70%です(maskクラス自身の高さの70%ではありません。)。boxクラスは30vhなので、21vh下にずれることになりますね。

同じく`left: 10px;`によって、boxクラスの左から10px、maskクラスが右にずれます。これは絶対値なので分かりやすいですね。

そしてこれを、「hoverした時に、`top: 0;`にしてあげる」CSSを書けばこのアニメーションは完成です。

positionプロパティって何なの?

以上

以下はアニメーションというよりpositionプロパティの話になります。

さて、ここでpositionプロパティについて説明します。positionプロパティはそのボックスの配置方法を決めるもので、相対位置(relative)か絶対位置(absolute or fixed)かを指定できます。

まずは絶対配置から説明します。子要素であるmaskクラスにabosoluteを指定しましたが、これでmaskクラスは絶対位置を基準として配置場所が決定されることになります。これはどういうことかというと、「ある基準点（絶対位置）を定めて、配置位置を決めますよ」という意味です。

「じゃあその基準点はどこになるの？」という所ですが、absoluteが指定されている場合、親要素をたどっていき、position: static;以外（relative、absolute、fixed）が指定されている要素を基準点とします。もしどこにも記述がなければ、ブラウザ(html要素)が基準点になります。

今回は親要素であるboxクラスに`position: relative;`を指定しているので、boxクラスが基準になります。つまり下図で言うと、赤枠で囲っている部分からどれだけずらして配置する？を指定することになります。
